| 보안 정보 이벤트 관리 | SIEM                      | 통합 로그 분석      | Splunk                       | 이벤트 상관분석                |
| 컨테이너 취약점 스캔  | Container Scanner         | 컨테이너 이미지 취약점  | Trivy, Clair                 | Docker, 이미지 스캔          |
| 디지털 포렌식      | Digital Forensics         | 증거 수집/분석      | Autopsy, FTK, EnCase         | 증거, 포렌식                 |
| 패스워드 크래킹     | Password Cracking         | 패스워드 복구/분석    | John the Ripper, Hashcat     | 해시 크랙, 패스워드             |
| 방화벽          | Firewall                  | 패킷 필터링/차단 방화벽 | iptables, pf                 | 패킷 필터, 접근제어 방화벽         |
| 웹 방화벽        | WAF                       | 웹 공격 차단       | ModSecurity, AWS WAF         | 웹 필터링, SQL인젝션 차단        |
| 악성코드 분석      | Malware Analysis          | 악성코드 동적/정적 분석 | Cuckoo Sandbox               | 샌드박스, 악성코드              |
| 무결성 검사       | File Integrity Monitor    | 파일 변조 탐지      | Tripwire, AIDE               | 파일 무결성, 변조 감지           |
