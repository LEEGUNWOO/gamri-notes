### 📌  Feistel 구조 vs SPN 구조

| 구분                    | Feistel 구조                                                                        | SPN 구조                                    |
| --------------------- | --------------------------------------------------------------------------------- | ----------------------------------------- |
| 구조 방식                 | 평문을 좌(L0) **(XOR처리)**, 우(R0) **(그대로 내림)** 로 나누어 처리 후 <br>교차하여 최소 2라운드를 거쳐야 모두 암호화 | Substitution Layer + Permutation Layer 반복 |
| 연산 방식                 | 함수 F 적용 후 XOR 연산으로 좌우 교환                                                          | S-box 치환 + P-box 순열                       |
| Confusion / Diffusion | 동일 수준 확보 위해 SPN보다 더 많은 반복 필요(2배)                                                  | 적은 반복으로 Confusion / Diffusion 달성          |
| ==**병렬 처리**==         | ==**어려움**==                                                                       | ==**가능**==                                |
| ==**복호화**==           | ==**암호화 구조와 동일 (복호기 별도 불필요)**==                                                   | ==**복호화 시 별도 역연산 필요**==                   |
| ==**특징**==            | ==**구조 단순, 구현 용이**==                                                              | ==**구조 복잡, 성능 우수**==                      |
| ==요약==                | ==XOR + 좌우 교환 구조, 복호화 동일==                                                        | ==치환 + 순열 구조, 복호화 별도 필요==                 |
| **알고리즘**              | **DES, SEED, CAST, Blowfish, Twofish, RC5, RC6**                                  | **IDEA, Rijndael(AES), ARIA**             |
