# 📘  NoSQL 기본 개념

## 1️⃣ NoSQL이란?

> RDBMS의 ACID 중심 구조 대신
> 분산 환경에 적합한 유연한 데이터 모델 사용

✔ 대용량
✔ 분산 처리
✔ 스키마 유연성
✔ 오픈소스 저렴
✔ Scale Out 용이 분산형이니
✔ 중복데이터를 여러 노드에 가지고 있어 가용성이 높고 오류 회복이빠름
✔ 범용적이지 않음,.특정조건에서 사용하거나 특정용도로 사용

#  2️⃣ NoSQL 유형

| 유형            | 특징      | 예시              |
| ------------- | ------- | --------------- |
| Key-Value     | 단순 키-값  | Redis           |
| Document      | JSON 문서 | MongoDB         |
| Column-Family | 컬럼 기반   | Cassandra Hbase |
| Graph         | 관계 중심   | Neo4j           |
