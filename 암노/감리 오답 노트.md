# 3. 3단계 감리 업무

|단계|사업자 제출|감리법인 점검|감리보고서|시정조치|
|---|---|---|---|---|
|**요구정의 단계**|요구사항정의서 + 과업내용 반영 대비표|계약문서 과업내용의 **요구사항 반영 여부 점검**|감리수행결과보고서|사업자 시정조치 → 감리법인 **시정조치확인보고서**|
|**설계 단계**|설계 산출물 + 과업내용 반영 대비표|설계 산출물의 **과업내용 반영 적정성 및 적합·부적합 판정 가능 여부 점검**|감리수행결과보고서|사업자 시정조치 → 감리법인 **시정조치확인보고서**|
|**종료 단계**|최종 산출물 + 과업내용 반영 대비표|세부 검사항목별 **과업 이행 여부 점검 (사업 검사 이전)**|감리수행결과보고서 (적합/부적합)|사업자 시정조치 → 감리법인 **시정조치확인보고서**|

| 번호  | 역할                                             |

| 2   | **과업범위(요구사항) 구체화 및 과업변경 영향·타당성 검토**            |

| 6   | **쟁점사항 기술검토 및 의견조율 지원**                        |
| 7   | **발주자 의사결정 지원 및 자문**                           |

# FTR (File Type Referenced)

## 정의

> **트랜잭션 기능(EI, EO, EQ)이 처리 과정에서 참조하거나 유지하는 ILF 또는 EIF**

(IFPUG Function Point Counting Practices Manual)

즉

- **트랜잭션 기능이 접근하는 논리 파일**
    
- **ILF 또는 EIF**
    
- **데이터를 읽거나 수정하는 파일**
    

# FTR 적용 대상

FTR은 다음 **트랜잭션 기능에서만 사용된다.**

|기능유형|FTR 사용|
|---|---|
|EI (External Input)|사용|
|EO (External Output)|사용|
|EQ (External Inquiry)|사용|

다음에서는 **사용하지 않는다.**

|기능유형|측정요소|
|---|---|
|ILF|RET + DET|
|EIF|RET + DET|

| EO  | FTR + DET |
