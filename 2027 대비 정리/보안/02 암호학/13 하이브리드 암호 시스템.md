## 📘 하이브리드 암호 시스템

### 📌 개념

- 메세지는 성능 좋은 대칭키로 대칭키는 공개키로 암호화 하는 두가지를 같이쓰는 하이브리드 (기밀성)
- 메세지 암호 < 대칭키로 성능달성 ,  키전달 (안전하게 공개키)
- 활용 사례: 전자봉투, PGP 암호화, SSL/TLS 암호 프로토콜

| 목적   | 설명                            |
| ---- | ----------------------------- |
| 성능   | 대칭키가 빠르니까 메시지는 대칭키로 암호화       |
| 키 전달 | 대칭키(세션키)를 안전하게 전달하기 위해 공개키 사용 |

### 📌 전자봉투(Digital Envelope)

- 메시지를 암호화한 **비밀키를 수신자의 공개키로 암호화**한 것
