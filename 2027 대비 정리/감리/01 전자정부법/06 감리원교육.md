## 📘감리원교육 

| 교육종류     | 대상                                  | 시간               |
| -------- | ----------------------------------- | ---------------- |
| 기본교육     | 감리원 등급 인정받으려는 경우                    | 40시간 이상          |
| 전문교육     | 수석감리원 등급 인정받으려는 경우                  | 50시간 이상          |
| **계속교육** | **모든 등급 감리원 / 행안부장관이 고시하는 사유 발생 시** | **3년마다 40시간 이상** |

### **📌〈계속교육〉**

| 구분            | 내용                                                                            |
| ------------- | ----------------------------------------------------------------------------- |
| **받아야 하는 사유** | 1. 감리원증 교부 후 **1년 경과**                                                        |
|               | 2. 법령·감리기준 개정에 따른 변경사항 발생 시 **위탁업무수행기관의 장이 판단**하는 경우 (경미한 사항은 교육자료 배포로 갈음 가능) |
| **해당하는 교육**   | 1. 위탁업무수행기관에서 실시하는 교육                                                         |
|               | 2. 정보기술 또는 감리관련 교육·세미나·심포지엄·포럼 등 위탁업무수행기관의 장이 인정하는 교육                         |
