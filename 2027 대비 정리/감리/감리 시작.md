| 유형        | 역할                      | 자격 요건                                                                                           |

| **전문가**   | 총괄감리원 책임하에 지원·자문        | 전체 감리인력의 **30% 범위 내**에서 AI·클라우드·빅데이터·모바일·정보보호·법률·회계 등 배치 가능                                     |

### 📌 감리대가 산정 예

① 감리기준의 3단계감리 보정 전 기본감리비 산정기준을 확인한다.

- 보정전기본감리비(3단계) = 0.9307 x IT감리 평균임금 x (감리대상사업비보정금액 / FP단가)^0.6385 x (1 + 제경비율) x (1 + 기술료율)

② 3단계감리 보정 전 기본감리비를 산정한다.

- ②-1 해당 연도의 SW기술자 IT감리 평균임금을 확인한다. (예. 2022년 적용 IT감리 직무 평균임금 424,481원)
- ②-2 해당 연도의 FP단가를 확인한다. (예. 기능점수 단가를 553,114원으로 적용) → 2024년 605,784원으로 조정됨(SW사업 대가산정 가이드)
- ②-3 제경비율을 결정한다. 제경비율은 110% ~ 120%의 범위 내에서 조정한다. (예. 110%로 적용)
- ②-4 기술료율을 결정한다. 기술료율은 20% ~ 40%의 범위 내에서 조정한다. (예. 20%로 적용)
- ②-5 감리대상사업비 보정금액을 산정한다.
    - 보정금액은 감리대상 사업의 예산에서 **부가가치세(10%)를 제외한 금액**에 감리대상사업비 구성항목별 보정비율을 곱한 후 합산한다.

|소프트웨어 개발비|하드웨어, 소프트웨어 구입비 및 유지보수비|행정정보 데이터베이스 구축비|전산실 공사비|사업비 총액(예가)|
|---|---|---|---|---|
|30억 원|10억 원|2억 원|1억 원|43억 원|

- 감리대상사업비보정금액
    
    - **EXAMPLE** = (30억 원/1.1) x 1.0 + (10억 원/1.1) x 0.456 + (2억 원/1.1) x 0.422 + (1억 원/1.1) x 0.000 = **32.185억 원 (부가세 미포함)**
- ②-6 3단계감리 보정 전 기본감리비를 산식에 따라 산정한다.
    
    - 3단계감리 보정 전 기본감리비 = 0.9307 x IT감리 평균임금 x (감리대상사업비보정금액/FP단가)^0.6385 x (1+제경비율) x (1 + 기술료율)
    - **EXAMPLE** = 0.9307 x 424,481 x (3,218,545,454원/553,114)^0.6385 x (1+1.1) x (1+0.2) = **252,302,780원**

③ 감리기준의 3단계감리 보정 후 기본감리비 산정기준을 확인한다.

- 3단계 보정 후 기본감리비 산정 기준 : 보정 후 기본감리비(3단계) = 보정 전 기본감리비(3단계) × (1+ 난이도계수합)

④ 3단계감리 보정 후 기본감리비를 산정한다.

- ④-1 난이도 수준을 선택한다.
    - 신기술 적용 수준(감리 복잡성을 증가시키는 신기술 적용 여부), 감리현장 지역 다중성(감리활동이 수행되는 감리현장의 지역 수)
    - 예. 신기술 적용 수준 '보통' 선택 : 0, 감리현장 지역 다중성 '복잡' 선택 : 0.05
- ④-2 난이도별 계수값 합계를 보정 전 기본감리비에 곱한다. 보정 후 기본감리비 = 보정 전 기본감리비 x (1+난이도계수 합)
    - **EXAMPLE** 보정 후 기본감리비 = 252,302,782원 x (1+0.05) = **264,917,921원**

⑤ 직접경비를 산정한다. (예. 진단도구 사용료 1,000,000원)

⑥ 3단계감리 보정 후 기본감리비와 직접경비의 합계에 대한 부가가치세를 구한다.

- **EXAMPLE** (264,917,921원 + 1,000,000원) x 0.1 = **26,591,792원**

⑦ 최종적으로 감리대가를 산정한다. 감리대가 = 3단계감리 보정 후 기본감리비 + 직접경비 + 부가가치세

- **EXAMPLE** 264,917,920원 + 1,000,000원 + 26,591,792원 = **292,509,714원**

※ 위의 예에서 추가로 상주감리, 추가감리를 계획할 경우, 상주감리 대가 산정기준을 적용하여 추가산정한다.

- Σ(투입공수 x 해당 IT직무 평균임금) x (1 + 제경비율) x (1 + 기술료율)

| 전자정부법 시행령                                       | 정보시스템 감리기준      | 감리수행가이드    |

**[1] 요구정의단계 감리**

- 계약문서에서 정한 과업내용이 **요구사항정의서**에 적정하게 반영되었는지 점검
- 실시 시점 : **분석단계 완료시점** (요구정의단계가 별도 구분된 단계가 아니기 때문)

**[2] 설계단계 감리**

- 사업자가 제출한 **대비표**가 적합·부적합 판정을 할 수 있도록 **구체화되었는지 확인**
- 설계 산출물이 과업내용을 적정하게 반영하고 있는지 점검하여 감리수행결과보고서에 명시
- 실시 시점 : **설계단계가 완료된 시점** (반복적 개발의 경우 중요도 높은 영역의 반복 차수를 정하여 해당 차수로 수행이 적절)

**※대비표란**

- **요구사항추적표를** 바탕으로 개별 요구사항에 대한 설계결과를 반영한 문서
- 요구사항ID 단위로 설계내용과 검사대상 및 검사기준을 추적·점검할 수 있는 문서
- 검사기준 : **검사대상, 검사방법, 예상결과, 판정기준 등을 명시** (기능요구사항은 테스트시나리오로 구체화 가능)
- 검사기준 반영 방법 : **요구사항ID 단위로 직접 명시하거나, 별도 작성한 검사기준의 내용을 참조할 수 있도록 관련 정보 명시**

**[3] 종료단계 감리**

- 사업자가 제출한 과업대비표의 과업내용 이행 결과에 대하여 감리법인이 **감리대상 사업 검사 전까지 세부 검사항목별 과업내용 이행의 적합·부적합 점검 결과**를 감리수행결과보고서로 작성하여 제출
- 실시 시점 : ==**통합시험 이후에 시작하고, 시정조치 확인 기간이 검사 이전에 종료될 수 있는 시점**이 적합==
- 과업이행 확인은 통합시험 또는 연계시험 관점에서 수행 → 사업자가 통합시험을 완료한 이후 수행

### 📌 감리원 윤리강령 (선언문)

나는 정보시스템 감리를 수행함에 있어 감리원으로서의 자긍심과 **사명감을** 가지고 맡은 바 직무에 최선을 다하여 정보기술 발전에 기여할 것을 다짐하면서 다음과 같이 선서한다. 나는 감리원의 직무를 수행함에 있어, **공익의 달성**을 우선으로 한다. **객관성과 공정성**을 유지한다. **감리 독립성을 확보**하고 **준법성과 비밀에 대한 보안을 유지**한다. 지속적인 자기계발을 통하여 **전문성을 유지하고 전문가적 태도와 품위유지**를 위해 노력한다.

### 📌 A100. 감리시행여부 판단

#### 🔥 감리 대상여부 판단 절차

①-1 정보시스템 특성기준에 해당한다고 판단한 경우

- 소액사업(1억 원 미만)인 경우, → 감리 효과가 높다고 행정기관 장이 인정하는 경우 : 의무감리 대상

② 정보시스템 구축사업인 경우

- 사업비가 5억 원 이상인 경우 : 의무감리 대상

③ 정보기술아키텍처(EA), 정보화전략계획 수립(ISP), 정보시스템 개발 및 운영 등을 위한 사업의 경우 해당 행정기관 등의 장이 감리시행 필요성 판단 - 감리시행의 필요성이 인정되는 경우 : 의무감리 대상

#### 🔥 정보시스템 구축사업의 범위

- (용어정의) 정보시스템은 **'정보', '기기', '소프트웨어'**로 구성됨
- 정보시스템 구축사업은 정보시스템을 구성하는 각 요소(정보, 기기, 소프트웨어)를 구축하는 사업, 각 요소를 복합적으로 구축하는 사업을 모두 포함함
    - 각 요소를 구축하는 사업 예시
        - DB구축 사업(정보를 구축하는 사업), SW개발 사업(소프트웨어를 구축하는 사업), HW장비 구축사업(기기를 구축하는 사업)
    - 각 요소를 복합적으로 구축하는 사업 예시
        - DB 구축 및 SW개발 사업(정보 및 소프트웨어를 구축하는 사업)
- 재구축(재개발), 일부 구축(일부 개발) 등 정보시스템 재구축 사업도 요구정의, 설계, 종료의 단계 구분이 가능한 경우, 정보시스템 구축사업에 포함됨

### 📌 A300. 감리대가 산정

- **감리대가는 감리기준 '별표 1 감리대가 산정기준'에 따라 산정한다.**
- **다만, 감리대상사업의 사업계획 수립시 확정된 예산은 부가가치세가 포함되어 있으므로 감리대가 산정시에는 부가가치세를 제외한 금액을 적용한다.**
- ※ 감리비 산정 시 실제 계약금액이 아닌 감리대상사업의 추정가격을 적용하는 이유는 감리 대상사업에 대한 업체선정 과정에서 덤핑 수주 시, 감리 대상사업의 과업 규모 변동 없이 사업비(계약금액)가 감소하고 이에 따른 감리 대가 저하로 인해 부실감리의 원인이 될 수 있기 때문이다.
