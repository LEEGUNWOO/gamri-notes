## 60-응용: 무선/IoT/USN/RFID/NFC/802.11/VPN/VLAN/CDN/NMS/프록시/LB/DPI/LPWA

> [!note] 무선·근거리: 보통 L1+L2가 중심
> - 802.11 무선랜, 블루투스, 지그비, NFC, RFID, LPWA  
> - ISM 2.4GHz: Wi‑Fi(2.4), Bluetooth, Zigbee 등이 흔히 사용(국가별 전파 규정은 별도).

### 60.1 USN/사물인터넷 네트워크
- **구성**: 센서/태그 → 게이트웨이 → 네트워크 → 플랫폼/서비스
- **시험 포인트**: 구성요소/통신방식/표준(지그비·블루투스 등) 비교.

### 60.2 RFID / NFC
- **RFID**: 태그-리더 무선 식별(주파수 대역/거리/구성요소가 포인트)
- **NFC**: 근거리(수 cm 수준) 통신/결제 등 응용
- **함정**: RFID와 NFC를 같은 거리/용도로 단정.

### 60.3 802.11 무선랜
- **핵심**: CSMA/CA, 채널/간섭, AP 구조
- **주변**: 보안(WPA2/3), 2.4/5GHz 차이(범위 내면).

### 60.4 핸드오프
- **정의**: 이동 중 기지국/접속점 전환
- **계층**: L2(링크) + L3(이동성) 다계층으로 묻는 경우가 많음.

### 60.5 Overlay / IPTV / VoIP
- **Overlay**: 상위에서 가상 네트워크를 얹는 개념(터널링/캡슐화).
- **IPTV/VoIP**: 응용 서비스로 출제(지연/지터/QoS가 주변 핵심).

### 60.6 VPN / VLAN / CDN / NMS / Proxy / Load balancer / DPI
- **VLAN**: L2(802.1Q) “논리적 분리”
- **VPN**: 종류별 계층(요약: IPsec=L3, SSL=L7)
- **CDN**: 콘텐츠 분산(주로 DNS/HTTP 동작과 연결)
- **NMS**: 운영/관리(SNMP와 연결)
- **Proxy**: L7 중계
- **Load balancer/L4 스위치**: L4 중심(+ L7 LB도 존재)
- **DPI**: L4~L7 심층 검사

### 60.7 NAT / 네트워크 관리 명령어
- **NAT**: 위 L3 섹션 참조
- **명령어**: ping/traceroute/netstat/arp/nslookup 등은 “무슨 계층 문제인지”로 구분해 암기(범위 내면).
