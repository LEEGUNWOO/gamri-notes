## 📘  NAC (Network Access Control)

### 📌 개념 및 구성

| 구분    | 내용                                                                |
| ----- | ----------------------------------------------------------------- |
| **개념**    | **•사용자 단말이 NW 접근 시 단말기의 보안정책 준수 여부와 사용자 인증정보를 검사해 네트워크 접속을 통제하는 솔루션** |
| **구성 요소** | **•사용자 PC ↔ 인증/정책 관리 ↔ AAA서버 ↔ 보안정책 적용**                              |

### 📌 NAC 구분

|구분|내용|
|---|---|
|Agent-base|•단말에 agent를 설치하여 PC의 정책 적용 상태 점검<br>•단말의 정책 적용 상태 점검 용이|
|Agent-less|•단말에 별도 agent 설치 없이 네트워크 경로상에서 NAC 기능 수행<br>•트래픽 모니터링 가능|
