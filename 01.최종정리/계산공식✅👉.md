```id="cm08"
[컨퓨전 매트릭스 / Confusion Matrix]

■ 구조
=            예측+
= 실제+   TP     FN
= 실제-   FP     TN

■ 핵심
= 행=실제 / 열=예측

■ 트리거
= TP(정양) FP(오양) FN(오음) TN(정음)

■ 함정
= FN↔FP 위치 혼동

[핵심 지표 한줄]

■ 공식
= Precision(정밀도) = TP/(TP+FP)
= Recall(민감도,재현율) = TP/(TP+FN)
= Specificity(특이도) = TN/(TN+FP)
= F1 = 2·(Precision·Recall)/(Precision+Recall)
= Accuracy(정확도) = (TP+TN)/전체
= Error Rate(오류도) = (FP+FN)/전체

■ 트리거
= Recall=실제양성 / Precision=예측양성

■ 함정
= 분모 기준 혼동 (실제 vs 예측)
```

```id="gini05"
[지니지수 / Gini Index]

■ 정의
= 클래스 불순도(혼합 정도) 측정

■ 공식 / 구조
= 1 - Σ(p_i²)

■ 트리거
= CART / 분류트리 / 불순도

■ 함정
= 개수로 바로 계산 금지 → 반드시 비율 p로 변환

■ 비교
= 0=순수 / 값↑ → 더 섞임

■ 계산 흐름 예시
= 전체 10개, A=4, B=6
= p(A)=4/10=0.4, p(B)=6/10=0.6
= 🔥Gini = 1 - (0.4² + 0.6²)
= 1 - (0.16 + 0.36)
= 0.48
```

```id="assoc01"
[지지도 / Support]

■ 공식
= P(A∩B) = (A와 B 함께 발생 수) / 전체

■ 트리거
= 빈도 / 같이 나온 비율

■ 예시
= 전체 10, A∩B=3 → 3/10=0.3

■ 함정
= 🔥 단순 빈도 (조건 아님)

[신뢰도 / Confidence]

■ 공식
= P(B|A) = P(A∩B) / P(A)

■ 트리거
= A 나오면 B 나올 확률

■ 예시
= A=4, A∩B=3 → 3/4=0.75

■ 함정
= 🔥 분모는 A (헷갈림)

[향상도 / Lift]

■ 공식
= Lift = P(A∩B) / (P(A)×P(B))

■ 트리거
= 독립 대비 연관성

■ 예시
= A=4/10, B=4/10, A∩B=3/10  
= (3/10) / (0.4×0.4) = 0.3 / 0.16 = 1.875

■ 함정
= 🔥 1 기준 (1=독립 / >1 양의 연관 / <1 음의 연관)
```

```id="bayes01"
[베이지안 추론 / Bayesian Inference]

■ 공식
= P(A|B) = P(B|A) × P(A) / P(B)
  B->A의 신뢰도 = A->B의 신뢰도 * A의지지도 / B의지지도


■ 트리거
= 사후확률 / 조건 뒤집기  

■ 사후 확율
= B가 발생했을 때 A일 확률 (사후확률) P(B|A)

■ 예시  
 -> 드라마기사 중 연예기사 비율
= 전체 100, 연예(A)=30 → P(A)=0.3  
= 드라마(B)=20 → P(B)=0.2  
= P(B|A)=0.6  
  
  -> 연예기사중 드라마
= P(A|B) = (0.6 × 0.3) / 0.2  
= 0.18 / 0.2  
= 🔥0.9🔥
  
■ 해석  
= 드라마 기사 중 90%가 연예기사

■ 함정
= 🔥 P(A|B) ≠ P(B|A)
```

```markdown
[앙상블 / Ensemble Learning]

■ 정의
= 여러 모델을 결합해 성능 향상

■ 핵심
= 약한 모델 여러 개 → 강한 모델

■ 트리거
= 정확도 향상 / 모델 결합 / 다수결

■ 함정
= 단일 모델 아님

[배깅 / Bagging (Bootstrap Aggregating)]

■ 정의
= 데이터 샘플을 여러 번 뽑아 모델 독립적으로 학습 후 평균

■ 구조
= 복원추출 → 여러 모델 → 평균/투표

■ 특징
= 병렬 처리 가능
= 분산(Variance) 감소

■ 트리거
= 랜덤 샘플 / 독립 학습 / 병렬

■ 함정
= 순차 아님

[부스팅 / Boosting]

■ 정의
= 이전 모델의 오답을 보완하면서 순차적으로 학습

■ 구조
= 모델1 → 틀린거 강조 → 모델2 → 반복

■ 특징
= 순차 처리
= 편향(Bias) 감소

■ 트리거
= 가중치 / 오답 보완 / 순차 학습

■ 함정
= 병렬 아님

[랜덤 포레스트 / Random Forest]

■ 정의
= 여러 결정트리를 배깅 + 랜덤 feature 선택으로 만든 모델

■ 구조
= Bagging + Decision Tree + Feature 랜덤 선택

■ 특징
= 과적합 감소
= 안정적 성능

■ 트리거
= 트리 여러 개 / 랜덤 특징 / 숲(Forest)

■ 함정
= Boosting 아님 (Bagging 계열)
```

# [데시벨 (dB)]

■ 정의  
= 신호 세기가 몇 배 변했는지를 로그로 표현한 값

■ 공식  
전력(SNR): dB = 10 log(P2 / P1)  << SNR
전압/전류: dB = 20 log(V2 / V1)

■ 기호  
P1 = 처음 값  
P2 = 변화 후 값  
P2 / P1 = 몇 배 변화

■ 계산 규칙  
곱셈 → dB 더하기  
나눗셈 → dB 빼기

■ 암기값  
2배 → +3 dB  
10배 → +10 dB  
100배 → +20 dB  
1/2 → -3 dB  
1/10 → -10 dB

■ 예시  
20배 → 10배 × 2배 → 10 + 3 = 13 dB

■ 트리거  
배수, log, 신호 세기

■ 함정  
전력은 10log, 전압은 20log 혼동 금지
