## 📘 UTM (Unified Threat Management)

### 📌 개념 및 구성

| 구분  | 내용                                                                                       |
| --- | ---------------------------------------------------------------------------------------- |
| 개념  | **•방화벽·IPS·VPN 기능을 하나의 장비에 통합한 솔루션**<br>•복합적인 보안 위협의 변화에 신속 대응하기 위해 NW보안과 콘텐츠 보안을 통합한 기술 |
| 대상  | •주로 중소 규모 기업의 통합보안 장비로 사용                                                                |

### 📌 UTM 구성

|구성요소|설명|
|---|---|
|구성 요소|•PC, G/W, 관리시스템, 서비스센터|
|보안 기능|•방화벽, IDS, IPS, VPN, 안티바이러스, 안티스팸, NAC, P2P보안|
|관리 기능|•패치관리, 정책관리, 로그관리, 사용자관리, 환경정보 관리|

### 📌 UTM 단점

| 구분                      | 내용                              |
| ----------------------- | ------------------------------- |
| 성능 이슈                   | •복구 기능의 단일솔루션 통합에 따른 성능이슈 발생 가능 |
| **Single Point of Failure** | **•단일장비 장애 시 전체 기능의 마비 가능**         |
