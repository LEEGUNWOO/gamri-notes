## 📘 감리법인 조건 

**〈감리법인 등록 - 전자정부법 제58조, 전자정부법시행령 제73조〉**

| 구분       | 내용                                                                                    | 비고                                                 |
| -------- | ------------------------------------------------------------------------------------- | -------------------------------------------------- |
| **등록조건** | **재정 능력** : 자본금 1억 이상                                                                 | 감리원은 그 법인에 상시 근무하는 사람으로서 다른 감리법인에 소속되지 아니한 사람이어야 함 |
|          | **기술 능력** : 감리원 5명 이상 (1명 이상 수석감리원)                                                   |                                                    |
|          | **결격사유** : 임원이 아래에 해당되면 결격사유                                                          |                                                    |
|          | ▶ 피성년 후견인 또는 피한정 후견인                                                                  |                                                    |
|          | ▶ **등록이 취소된 감리법인 임원으로서 등록이 취소된 날부터 2년이 지나지 아니한 사람 ==(등록취소의 원인이 된 행위를 한 사람과 그 대표자)==** |                                                    |
