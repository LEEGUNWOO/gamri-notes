| **감리인력 구성**    | • 사업 위험도 및 특성을 고려한 분야별 감리원·특정분야 인력 배치 수준의 적정성 여부<br>※ 요구하지 않은 상주감리원·특정분야 인력 제안, 요구 투입공수 초과 제안은 평가 제외<br>• 상근감리원의 비율                  |

| **각 분야별 감리인력** | • 투입 감리인력(요구하지 않은 상주감리원·특정분야 투입인력 제외)의 담당분야 관련 업무 또는 감리수행경력 전문성 여부<br>• 투입 감리원(총괄감리원 포함)의 계속교육 이수실적(교육 종류 및 시간)                      |
| **기타 지원사항 등**  | • 감리수행절차 또는 감리보고서 품질향상을 위한 감리법인 차원의 체계적 교육훈련·품질관리 수행 여부<br>• 시험·진단·점검 자동화 도구 및 기법 제안의 적정성 여부<br>• 발주자가 요청한 기타사항(감리범위 등)에 대한 적정 제안 여부 |

# 📘 감리원 윤리 준수 요소 행동 지침

|윤리 준수 요소|설명|
|---|---|
|**공익의 달성**|국가와 사회의 이익을 우선하는 직무 수행을 하며, 감리원 개인의 이익을 위하여 공익에 반하는 행동을 하지 않는 것|
|**객관성과 공정성 유지**|직무 수행 시 객관적·합리적 근거에 따라 판단하며, 직무 수행의 공정성을 해칠 수 있는 일체의 행위를 배격하는 것|
|**준법성과 비밀에 대한 보안유지**|관계 법령, 제 규정 및 실무 기준에 따라 객관적·독립적 관점에서 직무를 수행하고, 인지한 정보에 대해서는 보안을 유지하며 누설하지 않는 것|
|**전문성 유지**|감리기술 전문성 확보를 위하여 지속적인 노력과 부단한 자기계발을 하는 것|
|**사명감과 품위유지**|전문가의 지식과 기준에 근거하여 감리를 수행하며, 감리 수행 시 복장과 언어예절 등을 통해 전문가적 품위를 유지하는 것|

# 📘 적정 사업기간 산정 

## 1️⃣ 산정 대상

```
적정 사업기간 산정 대상 = **SW 개발** 관련 과업 포함 사업
```

❌ 제외

- ISP, BPR 등 컨설팅
    
- 운영·유지관리
    
- HW 도입
    
- 상용 SW 단순 도입
    

## 2️⃣ 1억원 예외

```
원칙 → 과업심의위원회 심의
예외 → 1억원 이하 = 기관장 직접 산정 가능
```

## 3️⃣ 1인 생산성 (FP/MM)

|사업규모(FP)|1인 생산성|
|---|---|
|1,000 미만|19|
|1,000~2,000 미만|22|
|2,000~3,000 미만|24|
|3,000 이상|22|

```
1200 FP → 22
```

## 4️⃣ 1년 초과

```
적정 사업기간 1년 초과 → 장기계속계약 체결
(국가계약법 적용)
```

| ------------------- | -------------------------------------------------------------------------- |

| **최종 감리대가**         | 보정후 기본감리비 + 상주 및 추가감리비 + 직접경비 + 부가가치세                                      |
 📌 **감리대상사업비보정금액는 부가세 제외**
 
제경비 = 직접인건비 × 110~120%  **(대가산정가이드에서는 144~154%)**
기술료 = (직접인건비 + 제경비) × 20~40%

- 제경비는 간접운영비
    
- 기술료는 기술사용·축적 대가

|**공통표준**|공통표준용어 + 공통표준단어 + 공통표준도메인|**“공통”은 용·단·도**|
|**기관표준**|기관표준용어 + 기관표준단어 + 기관표준도메인 + 기관표준코드|기관 내부 표준(코드 포함)|
|**DB표준**|DB 표준용어 + DB 표준단어 + DB 표준도메인 + DB 표준코드|개별 DB 적용 표준(코드 포함)|

> ✅ **함정 포인트:** “공통표준”에는 **코드가 안 들어간다**(공식 정의가 ‘용·단·도’).

## 🧾 3) 구성요소(정의) — 표준 구성요소를 ‘정확한 명칭’으로

|구성요소|공식 정의(요지)|키워드|
|---|---|---|
|**표준용어**|공공DB에서 업무적으로 쓰는 용어의 표준 정의|사용자 간 의사소통|
|**공통표준용어**|행안부장관이 정한 표준용어|“행안부장관”|
|**공통표준단어**|공통표준용어를 구성하는 단어 표준|최소 단위(단어)|
|**공통표준도메인**|데이터 값의 **형식**과 **값의 영역** 표준|형식/범위|
|**기관표준코드**|기관 업무를 기준으로 코드값/의미를 정해 활용하는 코드|코드값/의미/유일성|
|**DB표준**|(DB 표준용어/단어/도메인/코드) 통칭|DB단 적용|

---

## 2️⃣ 계산 구조 ⭐

  
     **지체상금 = 계약금액 × 지체상금률 × 지연일수**  


- 지연일수 = 계약 완료기한 다음 날부터 실제 완료일까지
    
- 계약서에 별도 비율 명시 시 그 비율 적용

|---|---|

📌 30% 숫자 기억
