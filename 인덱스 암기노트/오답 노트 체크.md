
# 정보보안 공격 기법 핵심 정리 (시험용 2페이지)

---

# 1️⃣ 신원 위장 및 세션 탈취 공격

**→ 공격자가 ‘다른 사용자나 시스템’으로 가장**

|공격|핵심 정의|특징|
|---|---|---|
|**Spoofing (스푸핑)**|신원 정보를 위조하여 정상 사용자처럼 행동|IP, ARP, DNS, Email 위조|
|**ARP Spoofing**|ARP 테이블을 조작하여 트래픽 가로채기|MITM 공격에 활용|
|**DNS Spoofing**|DNS 응답을 위조하여 가짜 사이트 연결|Pharming 공격에 사용|
|**Session Hijacking**|사용자 세션을 탈취하여 인증 우회|쿠키·세션ID 탈취|

📌 핵심 구조  
**Spoofing → MITM → Session Hijacking**

---

# 2️⃣ 네트워크 도청 공격

**→ 네트워크 트래픽을 몰래 수집**

|공격|핵심 정의|특징|
|---|---|---|
|**Sniffing (스니핑)**|네트워크 패킷을 캡처하여 정보 탈취|Wireshark 등 사용|
|**MITM (Man-in-the-Middle)**|통신 중간에서 데이터 도청 또는 변조|ARP Spoofing 등으로 수행|

📌 차이

- **Sniffing:** 단순 도청
    
- **MITM:** 도청 + 데이터 변조
    

---

# 3️⃣ 서비스 거부 공격 (DoS)

**→ 시스템을 마비시켜 정상 서비스 불가**

|공격|핵심 원리|
|---|---|
|**DoS**|단일 시스템에서 공격|
|**DDoS**|여러 시스템을 이용한 분산 공격|
|**SYN Flood**|TCP 연결 요청만 보내 서버 연결 큐 고갈|
|**Ping Flood**|ICMP 패킷 대량 전송|
|**Ping of Death**|비정상적으로 큰 ICMP 패킷 전송|
|**UDP Flood**|UDP 패킷 폭주|
|**Smurf Attack**|ICMP 브로드캐스트 증폭 공격|
|**LAND Attack**|송신 IP = 수신 IP|
|**TearDrop**|IP Fragment 재조립 오류 이용|

📌 핵심 특징  
**Flood 공격 = 대량 패킷 전송**

(RFC 4987)

---

# 4️⃣ 웹 애플리케이션 공격

**→ 웹 서비스 취약점 이용**

|공격|핵심 정의|특징|
|---|---|---|
|**SQL Injection**|입력값을 이용해 SQL 명령 삽입|DB 데이터 탈취·조작|
|**XSS**|웹페이지에 악성 스크립트 삽입|사용자 브라우저 공격|
|**CSRF**|사용자가 의도하지 않은 요청 전송|권한 악용|

📌 웹 공격 3대장  
**SQL Injection / XSS / CSRF**

(OWASP Top 10)

---

# 5️⃣ 사회공학 공격

**→ 기술이 아닌 사람을 속여 정보 탈취**

|공격|핵심 특징|
|---|---|
|**Phishing**|이메일·메신저로 가짜 사이트 유도|
|**Spear Phishing**|특정 개인을 대상으로 한 피싱|
|**Whaling**|CEO 등 고위 임원 대상 피싱|
|**Smishing**|SMS 메시지 이용 피싱|
|**Qshing**|QR 코드 이용 피싱|
|**Pharming**|DNS 조작으로 가짜 사이트 연결|

📌 차이

- **Phishing:** 사용자 속임
    
- **Pharming:** DNS 조작

# 6️⃣ 무선 네트워크 공격

|공격|정의|특징|
|---|---|---|
|**Evil Twin**|정상 AP와 동일 SSID의 가짜 Wi-Fi|MITM 공격 수행|
|**Rogue AP**|관리자 승인 없이 설치된 AP|내부 네트워크 침입|

---

# 7️⃣ 악성코드 (Malware)

|유형|정의|특징|
|---|---|---|
|**Virus**|프로그램에 기생하여 복제|숙주 필요|
|**Worm**|네트워크 통해 자체 확산|숙주 불필요|
|**Trojan Horse**|정상 프로그램으로 위장|사용자 실행 유도|
|**Backdoor**|인증 절차 없이 접근 가능한 통로|원격 제어 가능|
|**Rootkit**|관리자 권한을 숨기고 유지|탐지 회피|
|**Ransomware**|파일 암호화 후 금전 요구|데이터 인질|
|**Logic Bomb**|특정 조건에서 실행되는 악성코드|시간·이벤트 트리거|

(NIST Malware Classification)

# 8️⃣ 블루투스 공격

|공격|특징|
|---|---|
|**Bluejacking**|메시지 전송|
|**Bluesnarfing**|데이터 탈취|
|**Bluebugging**|기기 원격 제어|

📌 위험도  
**Bluebugging > Bluesnarfing > Bluejacking**
