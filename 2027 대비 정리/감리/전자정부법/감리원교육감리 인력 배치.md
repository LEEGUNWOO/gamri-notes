## 📘감리원교육감리 인력 배치 
(감리기준 제5조)

**전체 투입공수의 50% 이상 → 해당 감리법인 소속 상근 감리원으로 배치**

| 유형        | 역할                      | 자격 요건                                                                                           |
| --------- | ----------------------- | ----------------------------------------------------------------------------------------------- |
| **총괄감리원** | 감리대상사업 감리업무 총괄하는 상근 감리원 | 실제 감리 투입기간 **1년 이상**인 **수석감리원** + **상근감리원** (필수)<br>**(1년 기준은 단계감리 52주 상주감리1년등 실제 감리에 투입된 기간)** |
| **전문가**   | 총괄감리원 책임하에 지원·자문        | 전체 감리인력의 **30% 범위 내**에서 AI·클라우드·빅데이터·모바일·정보보호·법률·회계 등 배치 가능                                     |
| **상주감리원** |                         | 수석감리원(필수) + 아래 중 하나 해당                                                                          |
|           |                         | 1) 감리대상사업비 **20억 이상** 감리 참여 경력 **3회 이상**                                                        |
|           |                         | 2) PM 또는 QA 분야 경력 **3년 이상**                                                                     |
|           |                         | 3) 발주자가 적합하다고 인정한 수석감리원                                                                         |
